# Research Assistant Code Portfolio

This repository contains selected code that I developed during my position as a **Research Assistant at ETH Zurich**.

## Repository Structure

- **`Pixel/`**  
  Python and C++ code for pixel-related tasks, including charge injection procedures and automated S-curve studies built upon them.

- **`ChipQC/`** and **`DQM/`**  
  HTML and JavaScript code for readout and quality-control web interfaces.

## Scope and Limitations

The code in this repository was developed as part of a larger project and is therefore **not intended to be executed as a standalone package**.

For the same reason, supporting infrastructure and project-specific components are not included in this public repository. These include, among others:

- continuous integration workflows;
- watchdog applications;
- Docker environments; and
- other internal integration tools and dependencies.
